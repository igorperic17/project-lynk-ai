# Project Sidewalk

Project Sidewalk combines remote crowdsourcing + machine learning to map the accessibility of the world. Crowd workers virtually walk through cities labeling inaccessible areas. Collected data is used to generate new, accessibility-aware mapping tools and train machine learning algorithms.
