#  Evaluating the condition of marine resources in the Reserva Marina Tres Palmas, Rincon 

This project is part of National Coral Reef Monitoring Program of NOAA that aims to standardize reef and coral monitoring across all US coral reef jurisdictions. Even though this project is part of a National initiative, what makes this project different from others is that it includes a strong component of citizen participation in the collection and analysis of data.
